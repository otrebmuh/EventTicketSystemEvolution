# User Story: Email Verification (US002)

**As a** newly registered user  
**I want to** verify my email address  
**So that** I can access my account

## Acceptance Criteria

1. User receives a verification email with a verification link
2. Verification link expires after 24 hours
3. User can request a new verification email if needed
4. System shows clear success/error messages
5. After successful verification, user is redirected to the login page
