# User Story: Event Creation (US005)

**As an** event organizer  
**I want to** create a new event listing  
**So that** I can start selling tickets for my event

## Acceptance Criteria

1. Event organizer can access the event creation form from their dashboard
2. Event creation form includes:
   - Event name
   - Event description
   - Event date and time
   - Venue information
   - Ticket types and pricing
   - Event category
   - Event image
   - Maximum ticket quantity
3. System validates all required fields
4. System supports multiple ticket types per event
5. Each ticket type can have:
   - Price
   - Quantity available
   - Sale start/end dates
   - Per-person limit 
   - Venue zone
