# User Story: Notification Preferences (US023)

**As a** user or event organizer  
**I want to** manage my notification preferences  
**So that** I can control what notifications I receive

## Acceptance Criteria

1. User can:
   - Enable/disable specific notification types
   - Set quiet hours
   - Choose notification frequency
   - Update email address
2. System respects notification preferences
3. System provides preview of notification types
4. System maintains preference history
5. System allows bulk preference updates
