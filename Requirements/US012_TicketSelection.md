# User Story: Ticket Selection (US012)

**As a** user  
**I want to** select tickets for purchase  
**So that** I can attend the event

## Acceptance Criteria

1. User can select tickets from:
   - Event details page
   - Search results
   - Email promotions
2. Selection process includes:
   - Ticket type selection
   - Quantity selection
3. System shows:
   - Available quantities
   - Price per ticket
   - Total price
4. System validates ticket availability
5. System reserves selected tickets during checkout
