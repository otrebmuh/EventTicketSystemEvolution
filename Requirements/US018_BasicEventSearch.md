# User Story: Basic Event Search (US018)

**As a** user  
**I want to** search for events  
**So that** I can find events I'm interested in

## Acceptance Criteria

1. System provides a prominent search bar on all pages
2. Search supports:
   - Event name
   - Venue name
   - City/region
   - Date range
   - Category
3. System provides search suggestions
4. Search results show:
   - Event name
   - Date and time
   - Venue
   - Price range
   - Available tickets
   - Event image
5. Results can be sorted by:
   - Date
   - Price
   - Popularity
