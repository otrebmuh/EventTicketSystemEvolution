# User Story: Event Editing (US006)

**As an** event organizer  
**I want to** edit my event details  
**So that** I can update information before tickets go on sale

## Acceptance Criteria

1. Event organizer can edit event details from their dashboard
2. Editable fields include:
   - Event name
   - Description
   - Date and time
   - Venue information
   - Event image
3. System prevents editing of fields that would affect existing ticket holders
4. System maintains an audit log of changes
5. System notifies ticket holders of significant changes
