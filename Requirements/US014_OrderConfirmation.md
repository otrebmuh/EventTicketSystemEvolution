# User Story: Order Confirmation (US014)

**As a** user  
**I want to** receive confirmation of my purchase  
**So that** I know my tickets are secured

## Acceptance Criteria

1. System generates unique order numbers
2. System sends order confirmation including:
   - Order details
   - Ticket information
   - Payment confirmation
   - Receipt
3. User can view order history in their account
4. System supports order cancellation within specified timeframe
5. System handles partial order cancellations
