# User Story: Event Discovery (US020)

**As a** user  
**I want to** discover new events  
**So that** I can find events I might be interested in

## Acceptance Criteria

1. System provides event recommendations based on:
   - User preferences
   - Search history
   - Location
   - Popular events
2. Homepage displays:
   - Featured events
   - Upcoming events
   - Popular categories
   - Trending events
3. System supports event categories and tags
4. System provides related events suggestions
5. System shows events based on user's location
