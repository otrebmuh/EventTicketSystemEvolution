# User Story: Advanced Event Filtering (US019)

**As a** user  
**I want to** filter search results  
**So that** I can find exactly what I'm looking for

## Acceptance Criteria

1. System supports advanced search filters:
   - Price range
   - Date range
   - Category
   - Venue
   - Availability
2. Filters can be combined
3. System shows number of results for each filter
4. System remembers filter preferences
5. System provides clear/apply filter options
